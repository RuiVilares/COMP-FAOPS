You need to submit a dot file with the follow structure:

dinetwork {
   A[ color=blue, shape=triangle]
   B[ color=red, shape=circle]
   C[ color=red, shape=circle]
   A -> B [label="a"];
   A -> C [label="b"];
   B -> B [label="a"];
   B -> B [label="b"];
   C -> C [label="a"];
   C -> C [label="b"];
}
